## ■ コンテンツの更新方法

### お知らせを更新する
`src/components/News.tsx` を開き、`newsItems` の内容を編集します。

```
date:     "2026.06.01"    ← 日付
category: "新メニュー"    ← カテゴリ
title:    "新メニュー登場" ← タイトル
```

### メニューを更新する
`src/components/Menu.tsx` を開き、`menuItems` の内容を編集します。

```
name:        "料理名"
description: "説明文"
price:       "¥1,400"
tag:         "看板"
```

### 画像を差し替える
1. 新しい画像を `assets/original/` の該当フォルダに保存
2. `public/images/` の該当フォルダにWebP変換したファイルを配置
3. 各コンポーネントの `src` パスを新しいファイル名に変更

> 詳細は `docs/image-guideline.md` を参照してください。

## ■ 公開（デプロイ）について

現在のサイトはローカル環境（制作者のPC）でのみ動作します。  
一般公開するには **Vercel** でのデプロイをお勧めします。

### Vercelでの公開手順（概要）
1. [vercel.com](https://vercel.com) にアクセスしてアカウント作成
2. GitHubリポジトリ（`ryu-jin`）と連携
3. 「Deploy」ボタンをクリック
4. 数分で公開URL（例：`https://ryu-jin.vercel.app`）が発行される

> ※ Vercelの無料プランで公開可能です。独自ドメイン（例：`ryujin-ramen.com`）の設定は別途対応します。
