## 1. プロジェクト概要

### 目的
麺匠 龍神の世界観・こだわりをWeb上で発信し、新規来客の獲得および店舗認知度の向上を図る。

### ターゲット
- 立川エリア在住・在勤の30〜50代
- 本格ラーメンを求めるグルメ層
- SNS・スマートフォンで情報収集するユーザー

### コンセプト
> **「和モダン・高級感・静けさ」**  
> ラーメン店でありながら、一流割烹に通じる品格と落ち着きを持つWebサイト。  
> 訪れたユーザーが「行ってみたい」と感じる体験を提供する。

## 3. サイト構成

1ページスクロール型（シングルページアプリケーション）

```
[ヘッダー（固定ナビゲーション）]
      ↓
① Hero（ファーストビュー）
      ↓
② Concept（こだわり）
      ↓
③ Menu（メニュー）
      ↓
④ Gallery（ギャラリー）
      ↓
⑤ Staff（スタッフ紹介）
      ↓
⑥ News（お知らせ）
      ↓
⑦ Access（店舗情報・アクセス）
      ↓
[フッター]
```

## 4. 各セクション詳細

### ① Hero（ファーストビュー）
- 全画面キービジュアル
- キャッチコピー「一杯に込めた、龍神の魂」
- 「メニューを見る」「アクセス」の2つのCTAボタン
- PC・スマホ別に最適化した画像を配信

### ② Concept（こだわり）
- スープ・食材・店主の想いを3つのポイントで紹介
- 画像と文章の2カラムレイアウト

### ③ Menu（メニュー）
- 看板4品をグリッドで表示
- 料理名・説明・価格・タグ（看板/おすすめ/人気/限定）

| メニュー名 | 価格 | タグ |
|---|---|---|
| 龍神特製醤油ラーメン | ¥1,400 | 看板 |
| 黄金塩ラーメン | ¥1,350 | おすすめ |
| 濃厚味噌ラーメン | ¥1,450 | 人気 |
| 辛口つけ麺 | ¥1,500 | 限定 |

### ④ Gallery（ギャラリー）
- 5枚の写真をグリッドレイアウトで表示
- ホバー時に写真名を表示（龍神特製醤油ラーメン・店外・店内・カウンター席・厨房）

### ⑤ Staff（スタッフ紹介）
- 店主プロフィール・写真
- 引用文「魂を込めた一杯で、心を動かす。」

### ⑥ News（お知らせ）
- 新メニュー・イベント・営業情報を日付順で表示
- カテゴリバッジで視認性を確保

### ⑦ Access（店舗情報・アクセス）
- 住所・営業時間・定休日・電話番号
- Google Maps 埋め込み

**店舗情報:**

## 5. 技術仕様

| 項目 | 内容 |
|------|------|
| フレームワーク | Next.js 16（App Router） |
| 言語 | TypeScript |
| スタイリング | Tailwind CSS v4 |
| 画像形式 | WebP（高速配信） |
| レスポンシブ | PC・タブレット・スマホ対応 |
| SEO | メタタグ・OGP設定済み |
| ソースコード管理 | GitHub |

### 画像最適化
- FV: PC用（1920×1080px）・モバイル用（390×852px）を自動出し分け
- 全画像をWebP形式に変換（約25%の容量削減）
- Next.js の画像最適化機能を活用
